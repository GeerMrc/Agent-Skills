## 📖 本目录说明

本目录包含Frontend Design Agent Skills的质量标准和检查清单，确保代码质量、无障碍性、性能和可维护性符合高标准。

**核心价值**：
- 质量清单 - 完整的检查清单
- 审查标准 - 代码审查标准
- 测试策略 - 测试方法和覆盖率

**质量目标**：
- 代码覆盖率 > 80%
- WCAG AA 无障碍标准
- 移动优先响应式设计
- 性能优化（LCP < 2.5s）
- SEO最佳实践
